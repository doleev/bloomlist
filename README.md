# BloomList 🌷

A cute pink daily task list where finishing all your tasks for a day grows a flower into full bloom. Built with love.

- Per-day tasks with **To-Do** and **School** tabs
- Animated kawaii flower that grows as you complete tasks (seed → sprout → seedling → bud → blooming → full bloom 💕)
- **Week** and **Month** calendar views
- A garden of every day you've ever fully bloomed
- Streak tracking, sparkle confetti, sound effects, and an interactive flower (try petting it 7 times)

Single-file static HTML — no build step. Just open `index.html`.
